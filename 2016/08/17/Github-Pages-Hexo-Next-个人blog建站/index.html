Github Pages+Hexo+Next 个人blog建站

发表于 2016-08-17 14:53:19   |   分类于 技术 ， git   |  

https://hexo.io/zh-cn/docs/
https://xuanwo.org/2015/03/26/hexo-intor/
http://sunwhut.com/2015/10/30/buildBlog/
http://theme-next.iissnan.com/getting-started.html

下载主题：到github上找到next主题，下载zip文件后，解压到hexo安装目录中的themes下即可

本地启动
输入以下命令，可以在本地启动hexo进行测试工作
hexo s // hexo s -debug

关联到github
Hexo根目录下_config.yml文件，注意repo的url格式

打开.deploy_git/.git/config，注意

将其 deploy 到仓库中
打开 Git Shell 进入创建的文件夹，依次输入以下命令：

hexo next 主题添加多说评论
duoshuo.com 中注册账号
hexo\themes\next_config.yml添加key-value duoshuo_shortname: 多说域名前缀（也就是在注册多说时填入的域名前缀）

添加背景图片
首先找到一个背景图片放到 hexo（hexo工程文件）-> themes -> next -> source -> images 的路径下；
hexo（hexo工程文件）-> themes -> next -> source -> css -> _schemes -> Pisces（Mist和Muse也行），找到路径下的index.styl文件，在文件的最上方加上一代码body { background:url(/images/bg.jpg);} ，bg.jpg换成你的图片名

添加菜单及其图标
比如新加菜单projects 在hexo\themes\next_config.yml中

注意menu_icons中projects: cog 的cog是fontawesome中的图标名，可以从下面链接中查询图标以及对于的图标名，直接用即可，不需要下载
http://fontawesome.io/cheatsheet/

菜单中文可以修改hexo\themes\next\languages\zh-Hans.yml文件

如果你也觉得好，就值得
